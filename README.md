# shorty
links shortener

Сокращатель ссылок.

БД postgres in docker :
1. Длинная ссылка.
2. Короткая ссылка.

Эндпоинты:

- POST: запрос на создание ссылки:
  строка - длинная ссылка.
  В ответе короткая ссылка.
  
- GET: запрос по короткой ссылке,
  мы редиректим на длинную ссылку.

python version: 3.12 in docker
